# 🦸 Agent Avengers - 멀티에이전트 오케스트레이션 스킬

안녕하세요! 새로운 스킬을 만들어서 공유합니다.

## 뭐하는 스킬?

**agent-council + agent-orchestrator의 장점을 하나로!**

복잡한 태스크를 자동으로:
1. 서브태스크로 분해
2. 전문 에이전트 팀 구성
3. 병렬/순차 실행
4. 결과 통합

## 핵심 기능

### 4가지 에이전트 모드
- 🔷 **기존 에이전트** — 등록된 에이전트 활용 (watson, picasso 등)
- 🔶 **임시 스폰** — 일회성 서브에이전트
- 🟣 **멀티 프로필** — 다른 OpenClaw 봇 인스턴스 참여!
- 🔷🔶🟣 **풀 하이브리드** — 전부 조합

### 12가지 창발적 협업 패턴
- 🗳️ 경쟁 드래프트 — 여러 에이전트가 독립 수행 → 최고안 선택
- ⚔️ 적대적 협력 — Creator vs Critic 핑퐁
- 🧬 진화적 선택 — 솔루션 교배/진화
- 🐝 스웜 인텔리전스 — 마이크로 에이전트 다수
- 🪞 메타 관찰자 — 팀 코칭 에이전트
- ... 외 7개

### 포함된 스크립트
```
scripts/
├── assemble.py     # 태스크 분해 + 계획 생성
├── execute.py      # 실행 명령어 생성
├── monitor.py      # 진행 상황 모니터링
└── consolidate.py  # 결과 통합
```

## 사용 예시

```
"어벤저스 어셈블! A사, B사, C사 경쟁 분석해서 비교 리포트 만들어줘"

→ 자동으로:
  🔬 Researcher x3 (병렬 조사)
  🔧 Integrator x1 (결과 통합)
  📄 최종 리포트 전달
```
